# Student Motivation CrewAI

Проект для анализа академической успеваемости студентов и составления индивидуальных планов восстановления с помощью агентов CrewAI и Streamlit.

## 🚀 Установка

1. Clone repo:

2.
  ```bash
  git clone https://github.com/codeinveins123/student_motivation_crewai.git
  cd student_motivation_crewai
  python -m venv env
  # Windows
  env\Scripts\activate
  # Linux / Mac
  source env/bin/activate
  ```
3. Key
LLM_API_KEY=key

4. Run
For localhost:

  ```streamlit run streamlit.py```
  
or deploy on streamlit
